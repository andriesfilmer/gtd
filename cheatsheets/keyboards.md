## Skyloong GK61 PRO_48

* https://skyloong.vip/blogs/news/skyloong-gk61-pro-user-guide

* Open `chrome://device-log/`
* See which device has FILE_ERROR_ACCESS_DENIED
* If /dev/hidraw4, then `sudo chmod 777 /dev/hidraw4`
